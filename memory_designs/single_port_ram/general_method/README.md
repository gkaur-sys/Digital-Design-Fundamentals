# Single Port RAM (General Method)

This module implements a single-port RAM using behavioral Verilog modeling.

---

## 🧠 Features

* Synchronous read/write
* Write enable control
* Reset functionality
* 64 × 8 memory

---

## 🎯 Purpose

To understand how memory can be modeled manually in Verilog without relying on vendor-specific templates.
