# Digital Design Fundamentals using Verilog HDL

This repository documents my journey of learning digital system design using Verilog HDL, covering both basic and advanced concepts through hands-on implementation.

All designs are simulated using Xilinx Vivado, with selected modules synthesized on FPGA hardware.

---

## 📚 Repository Structure

* **Combinational Logic** – MUX, Encoders, Binary to 7-Segment
* **Sequential Circuits** – Counters, Shift Registers
* **Modeling Styles** – Behavioral, Structural, Gate-level, Switch-level
* **Adders** – Half Adder, Full Adder, Ripple Carry Adder
* **MUX Designs** – Hierarchical design (2:1 → 8:1)
* **FSM** – Mealy & Moore Machines
* **Memory Design** – ROM, RAM (IP-based)
* **Testbenches** – Simulation and verification

---

## 🛠 Tools & Technologies

* Verilog HDL
* Xilinx Vivado
* FPGA (Spartan Series)
* MATLAB (for ROM signal generation)

---

## 🎯 Objective

To build a strong foundation in RTL design by implementing digital circuits using multiple modeling styles and verifying them through simulation.
